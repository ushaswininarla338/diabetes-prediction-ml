# Diabetes Prediction using Machine Learning

This project predicts diabetes using the Pima Indians Diabetes Dataset with multiple ML algorithms.

## Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest
- KNN
- SVM
- Naive Bayes
- XGBoost

## Metrics Evaluated
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Dataset
Pima Indians Diabetes Dataset (768 rows, 8 features)

## How to Run
Open the notebook in Google Colab and run all cells in order.
